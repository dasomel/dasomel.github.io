안녕하세요! 기존 Jekyll 기반 블로그를 Astro로 마이그레이션 했습니다.

## 왜 Astro인가?

- **빠른 빌드 속도**: Jekyll 대비 훨씬 빠른 빌드
- **현대적인 개발 경험**: TypeScript, ESM 지원
- **Island Architecture**: 필요한 부분만 JavaScript 로드
- **Tailwind CSS 통합**: 간편한 스타일링

## 주요 변경사항

1. Jekyll → Astro 전환
2. Minimal Mistakes 테마 → 커스텀 Tailwind 디자인
3. Liquid 템플릿 → Astro 컴포넌트
4. Content Collections를 활용한 타입 안전 콘텐츠 관리

앞으로 Cloud, Kubernetes, DevOps 관련 기술 포스트를 꾸준히 작성할 예정입니다.
