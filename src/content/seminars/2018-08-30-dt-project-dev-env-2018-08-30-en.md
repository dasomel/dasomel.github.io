## Presentation Overview

This session in the Manufacturing On-Demand company-wide training course covers how to configure an efficient development environment for the development and operations of DT projects.

## Key Topics

### Overview of DT Project Development Environment
- Characteristics and requirements of DT projects
- Maintaining consistency between development and operational environments
- Core principles of environment configuration

### Development Environment Components
- Source code management (Git)
- Build and test automation
- Container-based development environment

### Operational Environment Configuration
- Deployment pipeline design
- Monitoring and logging framework
- Incident response and rollback strategy

### Hands-on Practice
- Step-by-step development environment setup
- Team collaboration workflow configuration

## Target Audience

- DT project developers in the manufacturing sector
- Participants in the On-Demand training course
- Teams considering integration of development and operational environments
