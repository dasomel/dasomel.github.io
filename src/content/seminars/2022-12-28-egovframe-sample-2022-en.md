## Presentation Overview

A hands-on seminar covering how to transition to a modern development environment using the e-Government Standard Framework's sample board.

**Host**: [eGovFrame Open Community (OPDC)](https://opdc.kr)

## Key Topics

### Kubernetes Deployment
- Containerizing sample board
- Kubernetes cluster deployment
- Service configuration and management

### SpringBoot Migration
- SpringBoot migration of existing projects
- Configuration file conversion
- Utilizing embedded server

### Gradle Migration
- Converting build system from Maven to Gradle
- Writing build scripts
- Dependency management optimization
