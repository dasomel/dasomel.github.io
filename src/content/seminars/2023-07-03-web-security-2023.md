## 발표 개요

OpenInfra Community Days Korea 2023에서 Kubernetes 기반 Keycloak과 OpenLDAP를 활용한 OpenID 기반 애플리케이션 간 인증/인가 구현 경험을 공유하는 세션입니다.

**일시**: 2023년 7월 3일

**트랙**: Track 2, Day 1

**시간**: 15:30 - 16:00

## 행사 소개

OpenInfra Community Days Korea는 한국의 오픈소스 인프라 및 클라우드 네이티브 기술 커뮤니티가 모이는 행사입니다. OpenStack Korea Community(2014~), Kubernetes Korea Group(2018~), Cloud Native Days 주최자들이 함께 만든 행사로 이틀간 다양한 기술 세션이 진행됩니다.

## 주요 내용

### 웹 보안 트렌드
- 최신 보안 위협 동향
- OWASP Top 10

### OpenID 기반 인증/인가
- Kubernetes 기반 Keycloak 구축
- OpenLDAP 연동
- 애플리케이션 간 SSO 구현

### 실무 경험 공유
- 실제 구축 사례
- 트러블슈팅 경험
- 보안 모범 사례

## 다루는 기술

- Cloud Native
- OpenInfra
- Ceph (Storage)
- Open Compute Project (OCP)
