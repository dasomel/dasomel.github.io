## Presentation Overview

A seminar covering how to build a cloud-based Docker development environment using OPDC 2.0.0.

**Target**: Web application developers and open community members

**Level**: Beginner

## Key Topics

### OPDC 2.0.0 Development Environment
An integrated toolkit applying cloud-based Docker technology:
- Eclipse (Development IDE)
- Jenkins (CI/CD)
- Nexus (Artifact Repository)
- Scouter (APM)
- SonarQube (Code Quality)
- SCM-Manager (Source Management)

### Environment Setup Practice
- Docker-based environment configuration
- Integrated development environment setup
- Practical application methods
