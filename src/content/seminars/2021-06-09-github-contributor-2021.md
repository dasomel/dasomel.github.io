## 발표 개요

2021 eGovFrame 컨트리뷰션을 통해 오픈소스 기여 문화를 소개하고, 실제로 표준프레임워크 컨트리뷰터가 되는 방법을 안내하는 실전 세미나입니다.

**주최**: [eGovFrame 오픈커뮤니티(OPDC)](https://opdc.kr)

**대상**: 관심 있는 누구나

**난이도**: 초급

## 주요 내용

### 빌드 환경 설정
- 개발환경 빌드 설정
- 실행환경 빌드 설정

### 컴포넌트 빌드
- 공통컴포넌트 빌드 환경 구성
- 템플릿 빌드 프로세스

### 실전 기여
- Pull Request 작성법
- 코드 리뷰 프로세스
- 기여 경험 공유
