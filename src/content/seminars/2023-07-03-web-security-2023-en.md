## Presentation Overview

A session at OpenInfra Community Days Korea 2023 sharing experience implementing OpenID-based authentication/authorization between applications using Kubernetes-based Keycloak and OpenLDAP.

**Date**: July 3, 2023

**Track**: Track 2, Day 1

**Time**: 15:30 - 16:00

## Event Introduction

OpenInfra Community Days Korea is an event where Korea's open source infrastructure and cloud native technology community gathers. It is an event created together by OpenStack Korea Community (2014~), Kubernetes Korea Group (2018~), and Cloud Native Days organizers, with various technical sessions over two days.
