## Presentation Overview

At the Open Community seminar, we introduce the collaboration between the Standard Framework Template project and K-PaaS, together with KakaoCloud.

## Key Topics

### Introduction to K-PaaS Lightweight Project v2.1

Introducing an upgraded project that provides a lightweight configuration (core features and installation sources) for running K-PaaS, now supporting various architectures including x86 and ARM.

### Key Features
- Lightweight configuration focused on core functionality
- Multi-architecture support for x86 and ARM
- Simplified installation sources for faster environment setup
- KakaoCloud integration strategy

### Standard Framework Template Integration
- eGovFrame template configuration running on K-PaaS
- Leveraging Standard Framework in cloud-native environments

## Target Audience

- Developers and operators interested in K-PaaS
- Teams looking to apply Standard Framework in cloud environments
- Those considering KakaoCloud service adoption
