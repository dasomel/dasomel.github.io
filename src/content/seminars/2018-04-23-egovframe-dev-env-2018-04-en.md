## Presentation Overview

A hands-on seminar to understand the structure of the Standard Framework and build an integrated development/operation environment applicable to actual projects through the OPDC 1.0.0 development environment.

**Target**: Web application developers and open community members

**Level**: Beginner

## Key Topics

### Understanding Standard Framework Structure
- Framework architecture overview
- Understanding the role of each component

### OPDC 1.0.0 Development Environment
An integrated development environment including the following tools:
- Eclipse (Development IDE)
- Jenkins (CI/CD)
- Nexus (Artifact Repository)
- Scouter (APM)
- SonarQube (Code Quality)
- SCM Manager (Source Management)

### Applying to Actual Projects
- Project environment setup methods
- Deployment demo demonstration
