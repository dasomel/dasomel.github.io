## Presentation Overview

A practical seminar introducing open source contribution culture through the 2021 eGovFrame Contribution program and guiding how to become a Standard Framework contributor.

**Host**: [eGovFrame Open Community (OPDC)](https://opdc.kr)

**Target**: Anyone interested

**Level**: Beginner

## Key Topics

### Build Environment Setup
- Development environment build settings
- Runtime environment build settings

### Component Build
- Common component build environment configuration
- Template build process

### Practical Contribution
- How to write Pull Requests
- Code review process
- Sharing contribution experience
