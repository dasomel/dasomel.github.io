## Presentation Overview

At the 2023 eGovFrame New Version Launch Event hosted by the National Information Society Agency, we present the session "Let's Try It! Understanding MSA Using Standard Framework Templates."

## Key Topics

### Introduction to the New Standard Framework Version
- Key changes in the 2023 new version
- New features and improvements
- Migration guide

### Understanding MSA Using Standard Framework Templates
- Understanding MSA (Microservice Architecture) concepts
- Structure of MSA templates based on the standard framework
- Hands-on MSA service development using templates

### Practical Application Guide
- Strategy for transitioning from monolithic to MSA
- Service decomposition principles and patterns
- Operational environment configuration and deployment

## Target Audience

- eGovFrame standard framework developers
- Public institution development teams considering MSA migration
- Engineers interested in cloud-native development
