## Presentation Overview

At the company-wide seminar, we introduce the internal Data Platform applied to defense industry and Ocean PoC projects. We share the platform's value through its background, architecture, real-world application examples, and a live demonstration.

## Key Topics

### Data Platform Background
- Changes in the data-driven business environment
- Current state and challenges of internal data management
- Need for a Data Platform

### Data Platform Introduction
- Platform architecture and components
- Data collection, storage, processing, and analytics pipeline
- Key features and characteristics

### Application Examples
- Defense industry PoC application case
- Ocean project PoC application case
- Results and expected outcomes

### Demonstration
- Real-time Data Platform demo
- Hands-on experience with key features

## Target Audience

- All employees interested in data utilization
- Data engineering and analytics personnel
- New project planning and development teams
