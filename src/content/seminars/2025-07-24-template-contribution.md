## 발표 개요

오픈커뮤니티 세미나에서 표준프레임워크 템플릿 프로젝트에 컨트리뷰션하는 방법을 소개합니다.

## 주요 내용

### 템플릿 Container 만들기

표준프레임워크 템플릿 프로젝트에 컨트리뷰션하는 전체 과정을 단계별로 설명합니다.

### 컨트리뷰션 준비
- 표준프레임워크 템플릿 프로젝트 구조 이해
- 개발 환경 설정
- GitHub 기반 컨트리뷰션 워크플로우

### 템플릿 Container 개발
- Container 구조 설계
- 표준프레임워크 컴포넌트 통합
- 테스트 및 검증 방법

### 컨트리뷰션 제출
- Pull Request 작성 가이드
- 코드 리뷰 프로세스
- 머지까지의 여정

## 대상 청중

- 표준프레임워크 오픈소스에 기여하고 싶은 개발자
- eGovFrame 생태계에 관심 있는 개발자
- 오픈소스 컨트리뷰션을 처음 시작하는 분
