## Presentation Overview

First training for IT project managers on spreading cloud native awareness in the public sector, organized by the Ministry of the Interior and Safety and the National Information Society Agency.

**Format**: YouTube Live

**Target**: IT project managers from central and local governments, public institutions

## Key Topics

### Considerations for Cloud Projects
- Cloud native adoption strategy for public sector
- Integration with existing systems

### Cloud Native Concepts
- Understanding containers and Kubernetes
- Microservices architecture

### Public Sector Application Cases
- Successful cloud transition cases
- Considerations for adoption
