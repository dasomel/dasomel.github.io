## Presentation Overview

Hosted by the National Information Society Agency (NIA), this collaboration seminar brings together the eGovFrame-based MSA template and the K-PaaS lightweight project. Hands-on exercises cover cloud-native application development, divided into backend and frontend tracks.

## Key Topics

### eGovFrame-based MSA Template
- Overview of eGovFrame MSA architecture
- Microservice decomposition strategy
- Utilizing standard framework components

### K-PaaS Lightweight Project
- Background and purpose of K-PaaS lightweighting
- Lightweight architecture configuration
- Deployment and operational environment setup

### Cloud-Native Application Development Hands-on
- Backend: Developing MSA services with Spring Boot
- Frontend: Developing SPA with Vue.js
- Deploying and integrating with K-PaaS environment

## Target Audience

- Developers using the eGovFrame standard framework
- Teams preparing for cloud-native migration on K-PaaS
- Engineers interested in MSA architecture
