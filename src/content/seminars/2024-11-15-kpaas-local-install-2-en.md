## Presentation Overview

Introducing how to install and operate K-PaaS locally on Apple Silicon (M1/M2/M3) Mac environment.

## Key Topics

### ARM64 Architecture Support
- Limitations of existing x86_64-based images
- Multi-architecture image build strategy
- Image distribution using Harbor registry

### K-PaaS Local Install 2.0.0
- Docker-based environment instead of Vagrant + VirtualBox
- Utilizing Kind (Kubernetes in Docker)
- Resource optimization and fast startup

### Hands-on Demo
- Installation process demonstration
- Key component verification
- Troubleshooting tips

## Target Audience

- Developers interested in K-PaaS
- Those who want to do cloud native development on Mac ARM environment
- Those interested in container-based local development environment setup
