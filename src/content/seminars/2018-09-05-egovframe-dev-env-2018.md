## 발표 개요

OPDC 2.0.0을 통한 클라우드 기반 Docker 개발환경 구축 방법을 다루는 세미나입니다.

**대상**: 웹 애플리케이션 개발자 및 오픈커뮤니티 회원

**난이도**: 초급

## 주요 내용

### OPDC 2.0.0 개발환경
클라우드 기반 Docker 기술을 적용한 통합 툴킷:
- Eclipse (개발 IDE)
- Jenkins (CI/CD)
- Nexus (아티팩트 저장소)
- Scouter (APM)
- SonarQube (코드 품질)
- SCM-Manager (소스 관리)

### 환경 구축 실습
- Docker 기반 환경 설정
- 통합 개발환경 구성
- 실무 적용 방법
