## 발표 개요

전사 세미나에서 방산, 오션 PoC 등에 적용된 사내 Data 플랫폼을 소개합니다. Data 플랫폼의 배경과 구성, 실제 적용 예시 및 시연을 통해 플랫폼의 가치를 공유합니다.

## 주요 내용

### Data 플랫폼 배경
- 데이터 중심 비즈니스 환경의 변화
- 사내 데이터 관리 현황과 과제
- Data 플랫폼 도입 필요성

### Data 플랫폼 소개
- 플랫폼 아키텍처 및 구성요소
- 데이터 수집, 저장, 처리, 분석 파이프라인
- 주요 기능 및 특징

### 적용 예시
- 방산 분야 PoC 적용 사례
- 오션 프로젝트 PoC 적용 사례
- 성과 및 기대 효과

### 시연
- Data 플랫폼 실시간 데모
- 주요 기능 직접 체험

## 대상 청중

- 데이터 활용에 관심 있는 전 임직원
- 데이터 엔지니어링 및 분석 담당자
- 신규 프로젝트 기획 및 개발 담당자
