## Presentation Overview

A hands-on seminar that builds on the theoretical foundation covered in the 90th seminar, actually setting up a local cloud test environment and deploying eGovFrame applications.

**Target**: Anyone interested (laptop and required programs must be prepared)

**Level**: Beginner

## Key Topics

### Cloud Concepts Introduction
- Review of basic cloud concepts
- Understanding Kubernetes architecture

### Hands-on Practice
**Local Cloud Test Environment Setup**
- Local Kubernetes cluster configuration
- Installing and configuring required tools

**Application Deployment**
- Containerizing eGovFrame applications
- Deploying to Kubernetes
- Service exposure and testing
