## K-PaaS Lite란?

K-PaaS Lite는 **K-PaaS 컨테이너 플랫폼을 간편하게 구축하고 테스트할 수 있도록 만든 경량화 버전**입니다.

## 개발 배경

- K-PaaS 컨테이너 플랫폼은 프로덕션 환경을 위한 완전한 기능을 제공하지만, 개발 및 테스트 환경에서는 과도하게 복잡하고 리소스를 많이 소비
- 개발자들이 K-PaaS를 쉽게 경험하고 학습할 수 있는 환경 필요
- Vagrant와 VirtualBox를 활용하여 누구나 쉽게 설치하고 사용할 수 있는 환경 제공

## 주요 특징

### 1. 간편한 설치
- Vagrant 기반 자동화된 설치 프로세스
- 단일 명령어로 전체 클러스터 구축
- 복잡한 설정 없이 즉시 사용 가능

### 2. 경량화된 구성
- 최소한의 리소스로 실행 (CPU 4코어, RAM 8GB)
- 개발 환경에 최적화
- 빠른 설치 및 구동 시간

### 3. 학습 및 테스트에 최적
- K-PaaS 컨테이너 플랫폼의 핵심 기능 제공
- 실제 프로덕션 환경과 동일한 구조
- 개발/테스트/데모 환경으로 활용 가능

K-PaaS는 한국지능정보사회진흥원(NIA)이 지원하는 국산 개방형 클라우드 플랫폼입니다.
